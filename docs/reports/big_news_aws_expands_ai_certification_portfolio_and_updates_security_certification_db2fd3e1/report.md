## 实施建议

### 推荐方案

本文主要面向希望获取AWS认证的专业人士，中国区用户可以：

- ✅ 正常注册并参加所有提到的认证考试（AWS Certified Generative AI Developer – Professional、AWS Certified Security – Specialty等）
- ✅ 访问AWS Skill Builder平台学习考试准备课程
- ✅ 参加Exam Prep Plan和练习评估
- ⚠️ 注意：涉及Amazon Bedrock的实践课程内容需要在全球区域环境中完成，或通过理论学习方式准备

**关键认证信息**：
- **AWS Certified Generative AI Developer – Professional**：Beta考试已开放注册，205分钟，85道题
- **AWS Certified Security – Specialty**：新版本SCS-C03已开放注册，旧版本SCS-C02截止日期为2025年12月1日
- **AWS Certified Machine Learning – Specialty**：将于2026年3月31日退役

### 替代方案

对于需要实践Amazon Bedrock的学习者：

1. **使用全球区域账号**
   - 实施方式：申请AWS全球区域账号进行Bedrock实践
   - 复杂度：低
   - 适用场景：有条件访问全球区域的学习者

2. **理论学习为主**
   - 实施方式：通过文档、视频教程和模拟题准备考试
   - 复杂度：低
   - 适用场景：主要目标是通过认证考试的学习者

### 风险提示

- **课程实践限制**：Advanced Generative AI Development on AWS课程中的Bedrock实践环节无法在中国区完成
- **认证有效性**：所有AWS认证均为全球通用，在中国区获得的认证与全球区域完全相同
