## 内容摘要

**文章类型**：新闻公告/合作伙伴计划发布

**主要内容**：

本文宣布AWS Resilience Competency（弹性能力认证）从服务合作伙伴扩展到软件合作伙伴。文章介绍了：

1. **背景说明**：在2023年re:Invent大会推出AWS Resilience Services Competency的基础上，现在扩展到软件合作伙伴领域

2. **认证分类**：AWS Resilience软件能力认证合作伙伴分为三个战略类别：
   - **弹性设计**：通过代理、负载均衡器、服务网格、集群解决方案等确保高可用性
   - **弹性运营**：通过可观测性、混沌工程和事件管理实现持续弹性
   - **弹性恢复**：专注于灾难恢复，最小化RTO和RPO

3. **启动合作伙伴**：展示了13家首批认证的软件合作伙伴，包括：
   - Arpio（灾难恢复）
   - Commvault（数据保护）
   - Cutover（恢复自动化）
   - Firefly（云自动化）
   - HPE Zerto（业务连续性）
   - NetApp（企业级存储）
   - Nobl9（可靠性平台）
   - PagerDuty（数字运营管理）
   - Rubrik（云数据管理）
   - SIOS（高可用性）
   - Solo.io（应用网络）
   - Veeam（备份解决方案）
   - vFunction（应用现代化）

4. **行动指引**：
   - 为客户提供了解合作伙伴解决方案的链接
   - 为合作伙伴提供了申请认证的指南
