### 主要内容

本文汇总了AWS re:Invent 2025大会（2025年11月30日至12月4日）的重要产品发布，涵盖以下领域：

**分析服务**
- AWS Clean Rooms 推出隐私增强的合成数据集生成功能，用于ML模型训练

**计算服务**
- AWS Lambda Managed Instances：结合无服务器简洁性与EC2灵活性

**容器服务**
- Amazon EKS 新增工作负载编排和云资源管理能力

**迁移与现代化**
- AWS Transform Custom：AI驱动的代码现代化服务
- AWS Transform Windows全栈现代化能力
- AWS Transform for Mainframe 重构能力和自动化测试功能

**网络与内容分发**
- Amazon Route 53 Global Resolver：安全的任播DNS解析（预览版）

**合作伙伴网络**
- AWS Partner Central 现已集成到AWS管理控制台

**安全、身份与合规**
- IAM Policy Autopilot：开源MCP服务器，简化IAM策略创建
