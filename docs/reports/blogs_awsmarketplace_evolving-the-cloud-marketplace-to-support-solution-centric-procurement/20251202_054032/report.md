### 主要内容

本文介绍了AWS Marketplace推出的多产品解决方案功能，该功能允许AWS合作伙伴将多个产品和服务组合成针对特定行业和客户用例的综合解决方案。

**核心要点**：

- **多产品解决方案能力**：合作伙伴可以将多个产品和服务打包成完整的解决方案
- **合作伙伴类型**：支持ISV、渠道合作伙伴、系统集成商和分销商创建解决方案
- **客户价值**：简化解决方案发现和采购流程，统一的采购体验
- **实际案例**：展示了Okta、SCC、Accenture、Westcon-Comstor、TEKsystems等合作伙伴的解决方案

**提及的AWS服务**：
- AWS Marketplace（核心主题）
- Amazon Connect（作为解决方案集成示例）
- Amazon QuickSight（作为解决方案集成示例）
