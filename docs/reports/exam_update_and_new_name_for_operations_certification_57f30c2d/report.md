### 评估说明

本博客主要介绍AWS认证考试的更新信息：

1. **认证更名**：AWS Certified SysOps Administrator – Associate 更名为 AWS Certified CloudOps Engineer – Associate
2. **考试更新**：考试代码从 SOA-C02 升级到 SOA-C03，新增容器、多账户架构、基础设施即代码等内容
3. **培训资源**：AWS Skill Builder 提供完整的考试准备计划

所有提到的培训和认证服务在中国区域均可正常使用，中国区用户可以：
- 注册并参加 AWS Certified CloudOps Engineer – Associate 考试
- 访问 AWS Skill Builder 获取考试准备资源
- 使用 AWS SimuLearn 进行实践练习
